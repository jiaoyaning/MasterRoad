## UiAutomator (测试多个应用的界面)
>UI Automator 测试框架提供了一组 API，用于构建在用户应用和系统应用上执行交互的界面测试。
>通过 UI Automator API，您可以执行在测试设备中打开“设置”菜单或应用启动器等操作。
